🔧 HOW TO BUILD AN IG PUZZLE GRID IN CANVA


✨ Option A: Create a Full Grid, Then Slice It


STEP 1: Start with a Large Canvas


Use this size to build a 3x3 grid (for 9 IG posts):


3240 x 3240 px (That’s 3 IG squares of 1080 x 1080 px)


If you want a 6x6 grid or more, use 6480 x 6480 px or adjust as needed.


STEP 2: Design Your Whole Puzzle as One Image


Add text, symbols (🜁, 🜄, etc), drawings, or collage-style photos.

Let elements flow across squares—this makes it feel interconnected.

Add faint gridlines or guides every 1080px to stay aligned.


STEP 3: Download the Image (PNG or JPG)


Once done, download the full square image.


STEP 4: Slice It Into 9 Squares (Free Tools)


Use:


PineTools Image Splitter

ImageSplitter.net

Or Photoshop / Preview if you have access


Choose to split by 3 rows x 3 columns. Then download the tiles!


Now each slice becomes one Instagram post—when uploaded in order, it forms the full puzzle map.



✨ Option B: Pre-Split Grid (For Organic Flow Posting)


If you want to post out of order (like you mentioned—grabbing a piece when inspired), here’s the method:


STEP 1: Build Each IG Post Individually (1080x1080)


Stick to one phase (e.g. “Awareness” puzzle pieces)

Use consistent elements: background, color, phase icon, and post number


STEP 2: Name Files Clearly


Example:

02_Air_WakeUp_Numbness.png


Then when people see multiple posts in the same category, they recognize they’re pieces of the same phase.
