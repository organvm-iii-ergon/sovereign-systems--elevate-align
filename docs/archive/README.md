# docs/archive/

The content genome. Everything Maddie shared, decomposed into atoms and analyzed.

## What Happened Here

Maddie provided 127 files — ChatGPT transcripts, voice notes, questionnaires — totaling ~360,000 words across health, mindset, business, astrology, and water science. We:

1. **Extracted** every distinct idea into an atom (1,821 total)
2. **Tagged** each with 17 metadata fields (provenance, nature, tier, pillar, editorial status, etc.)
3. **Verified** three ways (Claude, Gemini, Claude-blind) for structural consistency
4. **Graded** into SIGNAL (1,153 build-worthy), CONTEXT (557 structural), NOISE (111 filtered)
5. **Linked** 1,343 atoms to their GitHub issues
